# TASK-02: ステータスでのフィルタリング

## 概要
商品一覧で `active` / `inactive` による絞り込みをできるようにしてください。

## 要件
- [ ] 一覧画面にステータス選択UI（全て / 有効 / 無効）を追加する。
- [ ] API (`GET /api/products`) が `status` クエリを受け取れるようにする。
- [ ] Prisma の `where` 条件で絞り込む。
- [ ] `total` は絞り込み後の件数を返す。

## ヒント
- `status` が未指定なら全件を返す実装にすると後方互換性を保てます。
